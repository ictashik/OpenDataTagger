Style Add-ons (LoRA)

A style add-on nudges a base model toward a particular look — an art style, a character, a photo aesthetic — without needing a whole new model. They're small (usually under 200MB) and layer on top of whichever base model you pick. Once downloaded, each image tag in Define Columns can turn one on and set how strongly it applies.

To add one, search Hugging Face for a LoRA compatible with your base model and paste its repo ID below (format: username/repo-name).
